# WorldSample Project Page

Static academic project page for **WorldSample: Closed-loop Real-robot RL with World Modelling**.

## Local preview

From this directory, run:

```powershell
python -m http.server 8000
```

Then open `http://localhost:8000`.

The page uses only relative asset paths and can be deployed to GitHub Pages. Project media and the preprint are stored in `static/`.
